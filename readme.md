# alumni-platform
> Alumni platform for the University of Waterloo

## Installation

Clone the repository

```bash
git clone https://github.com/uwaterloo/alumni-platform.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
